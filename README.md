## Misc
### Regular Expressions for MOTD & Description values
Value | Setting
------------ | -------------
{{RegEx:.+}} | Match anything, has to be at least one character
{{RegEx:.*KEYWORD.*}}	| Matches the KEYWORD and anything before or after it.
